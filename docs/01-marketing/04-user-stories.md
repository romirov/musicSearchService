# Пользовательские истории

## 1. Поиск предложений для задачи пользователя

**В качестве** пользователя сервиса, \
**Я хочу** иметь возможность поиска по отрывку музыкальной композиции , \
**Для того, чтобы** знать название музыкальной композиции или ее автора

1. Сценарий: список предложений для моей потребности. \
   **Дано:** у меня есть отрывок музыкальной композиции. \
   **Когда** я загружаю отрывок музыкальной композиции \
   **тогда** я создаю топик, в котором зарегистрированные пользователи или сайт, \
   могут добавить в нить свои предположения либо точный ответ с названием композиции и артиста.

2. Сценарий: просмотр полного списка предложений. \
   **Дано:** у меня есть отрывок музыкальной композиции. \
   **Когда** я проигрываю сайту отрывок музыкальной композиции \
   **тогда** я создаю топик, в котором зарегистрированные пользователи или сайт, \
   могут добавить в нить свои предположения либо точный ответ с названием композиции и артиста.

## 2. Поиск потребностей для продукта пользователя

**В качестве** пользователя сервиса, \
**Я хочу** видеть запросы в виде топиков \
**Для того, чтобы** имееть возможность помогать другим пользователям сервиса.

1. Сценарий: список открытых топиков. \
   **Дано:** я вижу список открытых топиков. \
   **Когда** я захожу в открытый топик, \
   **тогда** я могу прослушать отрывок музыкальной композиции \
   **и** предложить ее название или имя артиста.

2. Сценарий: просмотр полного списка топиков. \
   **Дано:** я нахожусь на странице просмотра топиков. \
   **Когда** я нажимаю кнопку, \
   **тогда** я вижу полный список открытых топиков.
